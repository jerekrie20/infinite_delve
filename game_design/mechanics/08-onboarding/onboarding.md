# 08 · Onboarding

The first 30 seconds. **Decided** (D13 in [[DECISIONS]]): all 3 base classes
at creation, guided first run.

## The first session (target)

1. **Class select** — Squire / Archer / Apprentice, one screen, three
   silhouettes + one-line fantasies + the evolution chain teased
   ("Squire → Warrior → ???"). Pick is permanent for this climb; other
   classes come via restart later (frame it as "your first class")
2. **Guided first run** — the first delve IS the tutorial: 2-3 inline
   prompts only (auto-attack happens; tap ability when it glows; at depth 5:
   the Continue-or-Extract moment explained ONCE)
3. **Authored first-run script** — depths 1-5 use a fixed gentle seed:
   grunts, one guaranteed gear drop by depth 3, mini-boss at 5 beatable but
   scary
4. **First extract celebration** — banked screen: loot gleam, level-up,
   "you survived your first delve", frontier reveal ("your damage joined
   r/yoursub's war — boss HP ticks down on screen")
5. **Hook chain** — next-session reasons planted: Daily Delve badge,
   checkpoint progress bar toward depth 10

## Principles

- No text walls; teach at the moment of relevance
- **Skippable after prompt 1 (D35)** — a "skip tips" link on the first
  prompt; skipping never hides class-select or the first-extract
  celebration (those are the hook, not the tutorial)
- Everyone plays a real run — no auto-play demo
- The frontier cameo in run one is the community hook (blueprint: "your sub
  needs delvers")
- Time-to-first-dopamine < 60s (first kill + first drop)

## Current state
