# 11 · Torchrest — the home base (D43)

The between-runs screen becomes a PLACE. Every existing system gets a
building; every existing gold sink becomes a visible upgrade. Nothing here
adds a new progression currency — Torchrest *renders* progression the
player already earns.

## The buildings (v1: a side-view street, tap a building = its screen)

| Building | System it fronts | Visible upgrade tie |
|----------|-----------------|---------------------|
| **The Lift** | start delve, checkpoint picker, expedition dispatch | automation tiers = lift machinery visibly improves (rope → gears → engine) |
| **Forge** | crafting: reroll / affix / rarity upgrade / relic + pity-unique crafting (D44, D37, D46) · loot-filter rules (D47) | crafting unlock tiers = forge glows hotter |
| **Market** | consumable loadout purchase | stock expands with unlocked consumables |
| **Warehouse** | stash | stash pages = warehouse grows an annex per page |
| **Temple** | promotion quests + trial bosses (D36) | lit braziers per mastery earned |
| **Guildhall** | frontier board, Daily Delve, leaderboards, season | the Terror's banner hangs outside; damage bar on the wall |
| **Trophy Hall** | **the Codex (D45)**: bestiary + collection (unfound = silhouettes) + deeds; masteries, cosmetics, service record, season stamps | trophies/skins on display; completion % over the door — the public flex room |
| **Quarters** | class switching, loadout/rotation editing | current class at the window; mastered classes visible inside |

## Rules

- **No building timers, no build-currency** — Torchrest is navigation +
  celebration, not a second idle game. Upgrades happen instantly when the
  underlying purchase/milestone happens
- **Decoration = the monetization surface** (D2-safe): cosmetic building
  skins, lights, banners, pets wandering the street. Purchased OR earned
  (season rewards place free decorations)
- **Awards display**: Trophy Hall + street banners surface earned status
  publicly — screenshots of your street are the share artifact
- **The street darkens with the season**: as the Terror climbs the ladder,
  Torchrest's sky subtly dims (frontier state made ambient — grim-glow's
  version of a progress bar)

## Rollout

- v1 (with economy phase): Lift, Market, Forge, Warehouse functional as a
  street with tap-through panels (art can start as a single backdrop +
  hotspots)
- v1.5 (with chains phase): Temple, Quarters
- Launch: Guildhall, Trophy Hall, decorations
