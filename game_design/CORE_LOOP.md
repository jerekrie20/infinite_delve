# Core Loop (DELVE)

**Solo in the small, together in the large.** Your minute-to-minute is a solo
run; your day-to-day and season are shared with the whole sub. Decisions
referenced as D# live in [[DECISIONS]].

## 1. Session loop — one run (~2-10 min)

1. **Open** → welcome-back report (offline runs resolved — loot, gold, any
   deaths), hero, today's Daily Delve + frontier status
2. **Pick a checkpoint** → start at depth 1 or any unlocked boss checkpoint (D4)
3. **Fight** → auto-battle descends floor by floor; cast abilities (or let
   auto-cast handle them once purchased). Combat flows without interruption
   between decision floors (D3)
4. **Decide at checkpoints** → every 5th depth (mini-boss/boss floors):
   Continue deeper or Extract? A small flee button works anytime
5. **Resolve** → Extract banks the whole run (gear, gold, XP). Death loses
   everything unbanked since the starting checkpoint (D5). Hero persists
6. **Repeat** → push further, or switch to a saved mastered class (D8)

## 2. Daily loop

- **Daily Delve** — one shared seed, ONE attempt, real hero, per-sub depth
  leaderboard (D22). The appointment mechanic
- **Frontier damage** — every run today chips the sub's current frontier boss
  (D20); boss-kill days are event days
- **Offline runs** — set your policy before you leave (D19); tomorrow's open
  shows what your hero did overnight
- **Unlimited personal delves** — no energy, grind at will
- **Comments** — builds, brag runs, frontier coordination

## 3. Season loop (~4 weeks)

- The **frontier boss ladder** is the season objective — fell them all before
  reset (D20, D21)
- A **seasonal modifier** twists the rules each season (D21)
- **Sub vs sub** — subs ranked on frontier progress (D21, phased)
- Season end: boards archive, cosmetic rewards stamp, fresh ladder generates.
  Hero, masteries, and gear carry over — always (D21)
- One season ≈ one mastery chapter for an engaged player (D11)

## 4. Lifetime loop

- **The mastery collection** — every class branch fully evolved becomes a
  permanently playable saved class + account-wide bonus + inherited ability
  + status cosmetics (D7-D10). The roster IS the prestige ladder
- **Automation ladder** — the game gradually learns to play itself (D1);
  bought with gold, earned through play (D18)
- **Deep-push ladder** — endless depths (D6); deepest-ever is the forever stat
- Sub's history: frontiers conquered, titans felled, seasons archived

## Design principles

- **Tension first, automation earned** — the manual game must be fun BEFORE
  automation; automation is the reward for having mastered it
- **Banked is sacred** — nothing the player banked is ever taken away
- **Daily-equal + permanent-growth both safe** — instanced PvE, co-op
  competition (sub vs the world before player vs player)
- **Cold start safe** — fun with 3 players, better with 300
- **Never pay for power** (D2)
